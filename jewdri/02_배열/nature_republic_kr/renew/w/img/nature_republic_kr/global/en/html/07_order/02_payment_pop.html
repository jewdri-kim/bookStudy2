상품권/포인트 사용

모바일 상품권 모바일 상품권 선택 사용 가능 모바일 상품권 3개
포인트 P

*100P 단위 사용 가능 / 2,000P 이하 사용 불가

결제정보
무이자 할부 혜택 *법인/기업카드는 신용카드 무이자 할부 적용이 되지 않습니다.
공인인증 안내(30만원 이상 공인인증 의무사용) 안전결제 안내 안심클릭 안내
무통장입금(가상계좌) 방법
  1. 1 에스크로 약관동의
  2. 2 입금은행 선택
  3. 3 가상계좌번호 확인
  4. 4 주문금액 입금
  5. 5 결제완료
이용가능 은행 국민, 기업, 수협, 신한, 외환, 하나, 농협, 우리, 우체국 
보안서비스 의무적으로 에스크로서비스 적용
이점 무통장으로 가상계좌에 입금만 하면 자동적으로 입금확인이 됩니다. 
주의사항
  • - 각 결제 은행 가상 계좌로만 이용 가능합니다.
  • - 매 주문 시마다 새로운 계좌번호(개인전용)가 부여되며 해당 주문에만 유효합니다.
  • - 입금자와 결제금액이 정확히 일치할 때만 입금이 이뤄집니다.
  • - 주문 완료 후 7일 이내(주문일 포함)에 입금하지 않으면 주문이 취소됩니다.
실시간 계좌이체 안내
이용가능 은행 국민, 기업, 신한, 외환, 농협, 우리, 우체국, 제일은행 등 
이용 대상 인터넷뱅킹 이용고객 (공인인증서, 보안카드 필요)
이용 안내
  • - 이체 시 발생하는 수수료는 ㈜네이처리퍼블릭이 부담합니다. (無 수수료)
  • - 은행에서 직접 제공하는 보안 프로그램을 사용하여 고객정보가 유출되지 않습니다.
  • - 고객의 계좌/보안정보는 은행 내에 보관되므로 안심하고 거래할 수 있습니다.
  • - 실시간으로 결제, 입금확인이 되므로 배송처리가 빨라집니다.
주의사항
  • - 개인명의로 된 계좌로만 이용 가능합니다.
  • - 공인인증서가 있어야만 결제 가능합니다.
  • - 안전한 결제를 위하여 결제 도중 은행에서 제공하는 보안프로그램을 반드시 설치하시기 바랍니다.
  • - 결제 완료 후 1일이 지나면 자동 취소는 불가능하며 고객센터를 통하여 환불 문의해주시기 바랍니다.
휴대폰 결제 안내
휴대폰 결제란? 이동통신사, 휴대폰번호, 가입자 주민등록번호를 이용해 통신사 고객관리 시스템과 실시간 연동하여 인증하고 휴대폰으로 발송된 고유 승인번호를 결제창에 입력하는 휴대폰 결제 시스템입니다.
복합결제 안내

휴대폰 결제 시 기프티콘, 모바일 상품권, 적립금과 함께 사용하실 수 있습니다.

  • 1. 기프티콘 + 휴대폰 결제
  • 2. 모바일 상품권 + 휴대폰 결제
  • 3. 모바일 상품권 + 적립금(포인트) + 휴대폰 결제
이동통신사 SKT, KT, LG U+, 알뜰폰(헬로모바일, KCT)
주의사항 휴대폰 결제 취소는 결제 당월에만 가능합니다.
카카오페이
  • - 카카오톡 안에서 개인 신용카드를 등록 및 인증하여 카카오페이 비밀번호로 간편하게 즉시 결제하는 모바일 결제 서비스입니다.
  • - 본인 명의 스마트폰 카카오톡에서 본인명의 카드 등록 후 사용 가능합니다.

*현재, 신용카드 결제만 가능합니다.
