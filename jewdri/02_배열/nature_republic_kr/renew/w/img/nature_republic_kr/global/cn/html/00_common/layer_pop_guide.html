레이어팝업 공통가이드

  • - 레이어팝업 공통 스크립트 : /home/code/www/nature_republic/renew/w/js/ui.js
  • - 주석 : //레이어팝업(s), //레이어팝업(e) 사이
  • - 레이어팝업열고닫기함수 공통 : pop_layer_view(layer, st)
      >파라미터 layer: 해당레이어아이디값 문자열로,
      >파라미터st: 레이어를 열기/닫기 (열기는 null, 닫기는 'close' 문자열로)
  • - 레이어팝업열기 함수 : pop_layer_view(해당레이어아이디값);
  • - 레이어팝업닫기 함수 : pop_layer_view(해당레이어아이디값,'close');

레이어팝업 함수 호출 예시

- 레이어팝업열기

일반레이어팝업열기
<a href="javascript:pop_layer_view('#pop_type');">레이어팝업열기</a>
타이틀아래에텍스트있는레이어팝업열기
<a href="javascript:pop_layer_view('#pop_type2');">타이틀아래에텍스트있는레이어팝업열기</a>
알림레이어팝업열기
<a href="javascript:pop_layer_view('#pop_type3');">타이틀아래에텍스트있는레이어팝업열기</a>

- 레이어팝업닫기

* 아래 이미지의 레이어팝업 내 닫기버튼은 공통이벤트 (/home/code/www/nature_republic/renew/w/js/ui.js)




* 자동적으로 프로그램처리후, 레이어팝업을 닫을땐 공통닫기함수 사용 :


pop_layer_view('#pop_type','close');

(#pop_type 은 해당 레이어팝업의 아이디)

- 레이어팝업 마크업 위치



* 레이어팝업 마크업 위치는 #wrap 의 한단계 아래 자식에 넣기.

* 레이어팝업 마크업 위치는 #header, #container, #footer와 형제.

* 각 레이어마다, 식별할수 있는 id값 부여
